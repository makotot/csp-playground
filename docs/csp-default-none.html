default-src 'none' テスト

CSPポリシー

Content-Security-Policy: default-src 'none'

このページではすべてのリソースの読み込みがブロックされます。

結果

スタイルが適用されていない場合、CSPが正しく機能しています。

このページは素のHTMLのみで表示されているはずです。

ブロックされるもの

  • スタイルシート（インライン・外部）
  • JavaScript（インライン・外部）
  • 画像
  • フォント
  • その他すべてのリソース

確認方法

  1. ブラウザの開発者ツールを開く（F12キー）
  2. Consoleタブを確認
  3. CSP違反のエラーメッセージが表示されているか確認
