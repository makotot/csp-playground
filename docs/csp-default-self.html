🔒 default-src 'self' テスト

CSPポリシー

Content-Security-Policy: default-src 'self'

このページでは同一オリジン（'self'）からのリソースのみ許可されます。
インラインスクリプトとインラインスタイルはブロックされます。

インラインスタイルのテスト

このテキストは赤色で太字になるはずですが、インラインstyle属性がブロックされています。

インラインスクリプトのテスト

❌ インラインスクリプトがブロックされました

このメッセージが表示されている場合、CSPが正しく機能しています。

許可されるもの

  • 同一オリジンの外部スタイルシート
  • 同一オリジンの外部JavaScript
  • 同一オリジンの画像・フォント
  • head内の<style>タグ（このページで使用中）

ブロックされるもの

  • インラインスクリプト（<script>タグ内のコード）
  • インラインイベントハンドラ（onclick等）
  • インラインstyle属性
  • 外部ドメインからのリソース

確認方法

  1. ブラウザの開発者ツールを開く（F12キー）
  2. Consoleタブを確認
  3. CSP違反のエラーメッセージが表示されているか確認

期待されるエラー:
Refused to execute inline script because it violates the following Content Security Policy directive: "default-src 'self'"...
